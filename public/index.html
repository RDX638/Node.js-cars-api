Welcome to Cars API documentation

Cars Routes

Domain: https://netanelalroy.cyclic.app/

Method URL Body Description
GET /Cars Get a list of all the Car, will present 10 results per page
GET
  1. /Cars?page=2
  2. /Cars?perPage=4
  3. /Cars?reverse=yes
  4. /Cars?sort=price
    5. Note: You can use the above for all GET methods
     Note: You can also combined the above for multi filter option
     (/Cars?sort=price&reverse=yes&perPage=3)
  1. Get a list of all the car, will present 10 results in page 2
  2. Get a list of all the car, will present 4 results in page
  3. Get a list of all the car, will present 10 results in page in reverse order (default sort=year
  4. Get a list of all the car, will present 10 results in page, Defines what the list will be sorted by
GET /Cars/search?s={keyword(Name,Info)}
(/cars/search?s=cars)		 search for a car or a car description/info, will present 10 results per page
GET /Cars/category/{category name}
(/Cars/category/suv)		 Get a list of all the Cars in selected category, will present 4 results per page
GET /Cars/year?min={value}&max={value}
(/cars/price/?min=160000&max=1000000)		 Get a list of all the cars priced between 160000 and 1000000, will present 10 results per page (you can add only minimum year or maximum as well
/cars/price/?max=250000)
POST /Cars 
{
 "company": "Car company name",
 "model": "model",
 "year": yyyy,
 "category": "Category name",
 "price": price,
 "color": "color",
 "info": "Info about the car",
 "img_url": "image.jpg"
}
Add a new car to the data base, You need a token for this query
PUT /Cars/{_id} 
{
 "name": "car model name",
 "manufacturer": "manufacturer",
 "info": "model description ,
 "year": yyyy,
 "category": "Category name",
 "img_url": "img adress"
}
Update a car in the data base, You need a token for this query (only admin users alowed to modified other users entry)
DELETE /Cars/{_id} Delete a car from the data base, You need a token for this query (only admin users alowed to delete other users entry)
