Cars API Dcumentaion

Cars Routes

Method URL Body Description
GET /cars Get a list of all the cars,present 10 results per page
GET /cars/search search for a car or a car description, will present 10 results per page
GET /cars/?page=2 Get a list of all the cars, will present 2 results in page 2
GET /cars/category/suv Get a list of all the cars in suv category, will present 4 results per page
GET /cars/prices/?min=160000&max=1000000 Get a list of all the cars priced between 160000 and 1000000, will present 10 results per page
POST /cars 
{
    "company": "Mercedes-Benz",
    "model": "G-class",
    "year": 2018,
    "category": "SUV",
    "price": 850000,
    "color": "Black",
    "info": "Info about the car",
    "img_url": "image.jpg"
}
Add a new car to the data base, You need a token for this query
PUT /cars/1 
{
    "company": "Mercedes-Benz",
    "model": "G-class",
    "year": 2018,
    "category": "SUV",
    "price": 850000,
    "color": "Black",
    "info": "Info about the car",
    "img_url": "image.jpg"
}
Update a car in the data base, You need a token for this query
DELETE /cars/1 Delete a car from the data base, You need a token for this query
